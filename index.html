DataTags Interactive Visualizer

Instructions

You need to have a json file with the decision graph and tag space ready. You can generate one using the \toJson command in Tags' CliRunner. Or, you can download this demo file.

Please note that the demo has not been vetted for real-world use.

  1. START the application (this will open a new browser tab or window).
  2. Click the "open file" button
  3. A file selection dialog will appear. Select the json file and click "OK".

The DataTags Interactive Visualizer project was originally created by @tzulang for class "Topic in DataTags" at the CS department of BGU. Like the rest of the DataTags project, the Data Tags Interactive Visualizer is open-sourced and hosted at GitHub.
